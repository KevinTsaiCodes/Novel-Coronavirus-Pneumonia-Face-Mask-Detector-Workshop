# Environment Setup
### Download  [Anaconda](https://www.anaconda.com/products/individual)
### Create venv
### Open Anaconda Prompt/ Anaconda Powershell Prompt (Windows)
    conda create --name (venv-name) python=edition
### Open Terminal
    conda create --name (venv-name) python=edition
### Ex.
  conda create --name deeplearning python=3.7
